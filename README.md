# 英语句子写作

面向大一学生、以专四写作为目标的英语句子构造指南。

写出一个在专四考试中能够拿分的英文长句——骨架清晰、信息层次分明、语法合规、标点得当——需要哪些知识和技巧，全部内容分为三个部分：

- **Part I** 展示一个合格长句的样子，以及它是如何被造出来的。
- **Part II** 拆解构造这样一个长句所需的七项知识。
- **Part III** 把七项知识组装为一个可操作的五步流程。

所有规则说明使用中文，所有例句使用英文。语法术语在章内首次出现时以"中文（English）"的形式标注，并在[术语表](glossary)中汇总。

## 如何使用

**按顺序通读**。从 Part I 建立目标感，依次学习第 1–7 章，最后在 Part III 中把知识组装起来。完整阅读时间 6–8 小时。

**按失分点查阅**。模拟真题或批改作文时发现某类错误，直接跳转到对应章节，或查阅附录 A 的速查表。

**按构造步骤查阅**。写作过程中卡在某一步，直接查阅 Part III 的五步流程，并跳转到它引用的对应章节。

每一章的结构统一为：功能性清单开篇，规则 + 正例 + 反例 + 对比说明展开，失误速查表收尾。
